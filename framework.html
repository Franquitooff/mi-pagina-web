Modelo y Diagrama

Este es el sistema de gestión para un taller mecánico, desarrollado con Gineeri. Funciona tanto en Web como en WinForms, con una interfaz rápida, intuitiva y conectada a una base de datos local.

Probar demo

La arquitectura fue diseñada en PlantUML, dividiendo el sistema en capas: UI, lógica de negocio, persistencia y entidades. Gineeri permite que estas capas se conecten sin duplicar código.
